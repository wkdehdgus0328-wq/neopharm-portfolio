## ✅ 최종 전략 결정: "이미지 고정 + 텍스트 변주" 방식

### 왜 이 방식인가

1. **이미지는 "역량의 증거"** — HR이 읽지 않아도 "이 사람은 대시보드/코호트/퍼널을 다룰 줄 안다"가 시각적으로 전달됨
2. **텍스트는 "도메인 맞춤"** — 에스더포뮬러에 넣을 때는 "건기식 리텐션", CJ에 넣을 때는 "식품 재구매", 오설록에 넣을 때는 "프리미엄 F&B 충성도"로 문구만 변경
3. **가장 높은 비주얼 퀄리티의 이미지를 고정** → 기업별로 이미지를 바꾸면 매번 퀄리티 편차가 생김. 이를 방지

### 구체적으로

```
[고정 레이어] 이미지 3세트 (롬앤 기반) — 모든 기업 공통
[변주 레이어] 카드 제목, 서브 카피, KPI 수치 표현 — 기업별 맞춤
```

## 🖼️ 최종 이미지 선정 (전 브랜드 비교 후 확정)

### 카드 1: B2B SaaS 크로스채널 CRM 운영 구조

| 순위 | 후보 | 소스 | 판정 | 이유 |
|------|------|------|------|------|
| ✅ **확정** | 풀퍼널 플로우차트 | 롬앤 p2 | **채택** | 퍼포먼스→온사이트→CRM→데일리 전체 흐름이 한 장에. 4개 자료 중 유일하게 이 수준의 설계도를 가진 이미지 |
| 비교 | 캠페인 성과 비교 | 라이크유 p5 | 탈락 | Before/After 바 차트는 좋으나, "구조 설계 능력"을 보여주지 못함 |
| 비교 | 외부알림 주문 흐름표 | 지알엔 p5 | 탈락 | 라이크유와 동일한 한계 |

### 카드 2: 세그먼트 기반 캠페인 고도화 → ROAS 달성

| 순위 | 후보 | 소스 | 판정 | 이유 |
|------|------|------|------|------|
| ✅ **확정** | 캠페인별 ROAS 데이터 테이블 | 롬앤 p10 | **채택** | 타겟1(1,695%) vs 타겟3(125%) — "세그먼트가 성과를 결정한다"를 숫자로 증명. 가장 설득력 높음 |
| 비교 | 신규회원 캠페인 4차 체계 | 비건이펙트 p7 | 대안 보관 | 1차~4차 단계별 ROAS 테이블이 잘 나와있음. 건기식 타겟 기업에 텍스트 맞춤 가능 |
| 비교 | 스팟성 캠페인 세부 | 라이크유 p20-21 | 탈락 | 마인드맵 발송 이미지는 좋으나, 데이터 테이블이 없어서 밀도 부족 |

### 카드 3: 퍼널 이탈 구간 진단 → 이탈률 방어

| 순위 | 후보 | 소스 | 판정 | 이유 |
|------|------|------|------|------|
| ✅ **확정** | 코호트 분석 히트맵 | 롬앤 p15 | **채택** | "코호트를 직접 만들어 봤다"를 1초 만에 전달. 히트맵 비주얼이 4개 자료 중 유일 |
| 비교 | 장기간 이탈 세그먼트 바 | 지알엔 p15 | 대안 보관 | 365일 이상 이탈 64.8% 발견 — 건기식 이탈관리 스토리에 쓸 수 있음 |
| 비교 | 재구매 주기 곡선 | 비건이펙트 p50 | 대안 보관 | 66% 30일 이내 재구매 — 리텐션 마케팅 스토리에 강력 |

## 🎯 최종 확정: 3카드 구조 + 이미지 배치

### 카드 1: 크로스채널 CRM 오퍼레이팅

```
메인 이미지: 롬앤 p2 — 풀퍼널 플로우차트
서브 이미지: 롬앤 p9 — 세그먼트별 메시지 크리에이티브 (중장기이탈/관심/신규/장바구니이탈)
```

**텍스트 (도메인 변주 가능):**
- 기본: "40개 브랜드 병렬 운영, 퍼포먼스-온사이트-CRM 3채널 동기화"
- 에스더포뮬러 맞춤: "건기식 구독형 고객의 재구매 주기에 맞춘 자동 CRM 시퀀스 설계"

**HR이 읽는 메시지:** "이 사람은 CRM을 '발송'이 아니라 '설계'하는 사람이다"

### 카드 2: 세그먼트 기반 캠페인 → ROAS 985%

```
메인 이미지: 롬앤 p10 — 캠페인별 ROAS 상세 테이블
서브 이미지: 롬앤 p19 — 롬앤 vs 타사 성과 비교 + 목표 설정
```

**텍스트 (도메인 변주 가능):**
- 기본: "코호트 3개(신규/관심/이탈) 분류 → 피로도 3일 캡 → 타겟1 ROAS 1,695%"
- 에스더포뮬러 맞춤: "에스더 팩 App 사용자 세그먼트 × 구매 주기 매칭 → 재구매 전환 설계"

**HR이 읽는 메시지:** "세그먼트가 성과를 결정한다는 것을 데이터로 증명한 사람이다"

---

### 카드 3: 퍼널 이탈 진단 → 이탈률 12.5%p 방어

```
메인 이미지: 롬앤 p15 — 코호트 분석 히트맵
서브 이미지: 롬앤 p13 — 미구매 방문자 현황 (85~88%)
```

**텍스트 (도메인 변주 가능):**
- 기본: "미구매율 85% 진단 → Exit-Intent 3단계 넛지 → 이탈률 12.5%p 방어"
- 에스더포뮬러 맞춤: "건기식 초기 구독 이탈 구간(D+7~D+30) 분석 → 넛지 설계"

**HR이 읽는 메시지:** "문제를 정의하고 데이터로 풀 줄 아는 사람이다"
